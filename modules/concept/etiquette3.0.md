# 礼儀3.0
**礼儀3.0（Etiquette 3.0）** とは、礼儀のメンタルモデルであり「意思決定を示すのが礼儀だ」とするもの。

従来のメンタルモデルとして、[GOROman](https://x.com/GOROman/status/984872963655680000) による以下が存在した:

> 礼儀1.0：相手を重んじる。自分の時間を犠牲にし、時間を相手のために使う。直接会う。スーツなど服装をわきまえる。
> 
> 礼儀2.0：相手の時間を奪わないようにする(電話しない、リモートで済むものはリモート)

つまり礼儀 1.0 とは「差し出すのが礼儀だ」とするものであり、礼儀 2.0 とは、「差し出させないのが礼儀だ」とするものである。

これに続く第三のパラダイムが礼儀 3.0 であり、意思決定を示すことこそが礼儀だと考える。

具体的を挙げる:

- 自分宛のメッセージすべてに何らかの返信または反応を示す
- FYI に対し、必ず自分の見解をつける
- 人それぞれやケースバイケースの言い分に逃げず、現時点の持論を展開する
- 外部圧力や相手が折れるのを待つのではなく、否定の旨を速やかにはっきりと示す

また、意思決定は、しばしば形に残さねば有効性が続かないため、言い換えるなら「意思決定を残すのが礼儀だ」ということもできる。その意味で、読み書きや非同期コミュニケーション、また同期コミュニケーションであっても録画など記録の許容が要求される。これらに応じず意思決定を残さない者や、そもそも意思決定を示そうとしない者は、礼儀 3.0 で言えば無礼なのである。
