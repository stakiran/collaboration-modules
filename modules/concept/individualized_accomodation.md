# 個別的配慮
**個別的配慮（Individualized Accomodation）** は、合理的配慮の代替となる概念であり、ニューロダイバージェントを最初から特別扱いするという考え方である。

## 背景
元々障害者向けの概念として合理的配慮があったが、これは既存の規範に合わせるとの前提が第一にあって、その上で障害者向けにどこまで緩和するかを考えるメンタルモデルとなっている。また配慮事項検討のリードも障害者本人ではなく会社側、上司側、チーム側など定型発達側が行う。したがって配慮が足りないか、搾取になりやすい。

昨今ではニューロダイバシティの重要性も増している。手帳は持っていないがニューロダイバージェントの診断を受けているとか、同じニューロダイバージェントでも IQ の違いがあって一方は適用できるが他方は適応できないといったグラデーションが生じている。障害としても、身体的に見えるものではないため、ただでさえ配慮がしづらい。合理的配慮の実情では力不足である。

## 個別に配慮する
そこで個別的配慮の概念を使う。合理的配慮とは以下の点で異なる:

|  | 規範に合わせるのが第一 | 対象者個人の特性を踏まえるのが第一 |
| --- | --- | --- | 
| 定型発達が主導する | 合理的配慮 | - |
| ニューロダイバージェントが主導する | - | 個別的配慮 |

まず、対象個人の特性を踏まえることを第一とする。たとえば 8:30 ～ 17:30 勤務かつハイブリッドワークの会社に、その生活リズムでは仕事も生活もままならない ASD がいたとする。この者は自身のライフスタイルを最適化しており、朝型で生きていて、6:00 ～ 15:00 かつリモートメイン（出社は必要に応じて、月1以内が目安）なら逸脱しないとしよう。合理的配慮だと、8:30 ～ 17:30 への適応や、ハイブリッドワークへの適応をいかにさせるかが議論となるが、個別的配慮では逆にこの者の希望をどれだけ守れるかが議論となる。同じように見えて、違う。スタートラインが違えば、結果も異なったものになる。合理的配慮だと、おそらくこの者は朝型もリモートメインもどちらも手に入らないだろう。結果、明らかに合わないライフスタイルを敷いられ、当然仕事でパフォーマンスも出せずに衰退していく可能性が高い。合理的配慮では足りないのだ。

次に、配慮のリードについても、対象者たるニューロダイバージェントが自ら行う。なぜなら自身の特性と対処法を最も良く知るのは対象者自身だからである。定型発達側がよく使うのは社内の産業医や人事との相談、あるいは継続的通院による情報収集であるが、どちらも悪手である。ニューロダイバシティのあり方は千差万別であり、最もよく知る本人が主体となって配慮を検討するのが最も近道であることが多い。そもそも専門家や外部機関は一般論しか言えないが、我々が求めているのは、この組織このチームにおける具体的な配慮である。
