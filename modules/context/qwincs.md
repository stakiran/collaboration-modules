# QWINCS
**QWINCS** とは主なコミュニケーションツールを整理したアクロニムであり、以下の略:

- Q&A
- Wiki
- Issues（チケット型）
- Notes
- Chat
- Sticky boards（強引だがデジタルホワイトボード）

コミュニケーションツールというと Slack や Teams などチャットを思い浮かべるが、他にも 5 つ存在すると言っている。チャットしか知らない、使ってないのは機会損失であり、たとえば非同期コミュニケーションを満足に行うことはできないだろう。実際チャットしか知らない組織はオンラインミーティングや出社にも頼らざるを得なくなっている。

機会損失をやめるためにも、この 6 種類のツールはすべて試行し、どのように使えるかを思考したい。以降では推奨するメンタルモデル（向き合い方）を述べるが、これにとらわれなくても良い。

## Q&A
to C 向けには Yahoo! 知恵袋、Quora、Stack Overflow などが知られている。to B 向け、特に社内で社員向けに提供するものについてはブルーオーシャンであり、主力製品はない。理由は単純で、成立が難しいからである。

Q&A は誰でも質問と回答を行えるようにし、質問者が最良の回答（ベストアンサー）を選ぶことでクローズする。数の力で、質問者が知りたいことを知ることに特化する。その性質上、従業員数は数千以上が必要である。また、油断するとエンタメ色の強い雑談に支配されてしまうため、強い規律も求められる。to C 向けの Q&A も、上位トピックはニュース、ゴシップ、ポエムばかりである。

余談だが、チャットで手短に体験したければ、Rapid Q&A を使うこともできる。

## Wiki
ウィキは関係者全員が誰でも読み書きできる軽量かつ高速なウェブサイトであり、プレーンテキストによる記述とページ間リンクによるネットワーク化を特徴とする。pukiwiki や Mediawiki など歴史は古いが、近年では Cosense のような現代的なものも登場しており、ウィキの弱点であるページ内同時編集不可を突破している。

ウィキは実は強いガバナンスを求めるものであり、読者全員にとって役立つナレッジを整備する場所と言える。それゆえ整理整頓が最重要であり、容易に「見当違いの管理による形骸化」が起こり得る。整理整頓が不要な Cosense でさえも、人間の認知能力やクライアント側・サーバー側の性能の問題もあるため、無尽蔵に何でも書いていいわけではない。事実上、何を書くべきか、特に書かないべきかの規律は必要である。これをナレッジ指向と呼ぶ。

## Issues
チケットと呼ばれる類のツールであり、GitHub Issues を指すと考えて差し支えない。元は課題管理ツールであったが、汎用的なコミュニケーションツールとしても使えるし、実際 GitHub は Issues をベースとした Discussions もリリースしている。

Issues はトピック指向と言え、1つの話題を1つのIssue、もっというとページで扱うという独立性に価値がある。話題 A に関することはページ A に書けばいい。話題が済んだらクローズすればいい。新しい話題が生じたら、別にページをつくって、そこでやりとりするべきだ。これによりコミュニケーションは話題単位で構成される営みとなり、非同期的コミュニケーションがしやすくなる。Communication As A Task も参考にせよ。

## Notes
Notion、Google ドキュメント、Box Notes など一つのノートを同時編集できる体験は身近になった。QWINCS ではこれらをノートと呼んでいる。

ノートは実はバーチャルオフィスの役割が強い。ノートという名のワールドであり、カーソルという名のアバターなのだ。複数人が集まって、その場でコラボレーションできることに価値がある。そのためテキストのみならず画像や動画その他埋め込みなど抱負なメディアをサポートしていることが多い。体験が重要なのだ。ストック情報を整備するという観点は二の次でしかないし、おそらくウィキの方が向いている。

## Chat
Slack や Teams といったチャットを指す。LINE その他 SNS とは違って、ワークスペースやチャンネルやスレッドといった「部屋」の機能があり、これゆえにビジネス用途に耐えることができる。ビジネスチャットと呼ぶこともある。

Chat は時系列指向であり、単にメッセージを時系列で表示するだけである。人間にとってはわかりやすいが、それだけだ。これだけで仕事を完結できるものではない。特に今その場でやりとりをする際はシンプルで使いやすいが、その後が致命的に弱い。検索や共有や整理はほぼ確実に形骸化する。結局、非言語コミュニケーションでカバーするためにミーティングに頼らざるを得なくなる。

しかし、QWINCS を知っておれば、他のツールを使うことで非同期コミュニケーションの幅を広げることができる。QWINCS という概念の主目的は、Chat にとらわれている者を救うことだ。

## Sticky boards
きれいなアクロニムをつくるために強引に名付けたが、デジタルホワイトボードを指す。Miro が有名だ。

デジタルホワイトボードの本質も、実はバーチャルオフィスにある。ノートは一次元的だったが、ボードは二次元的である。マイクラや VRChat など三次元ほどの表現力はないが、ノートよりは高い。両者の中間の塩梅を取れるツールである。

指向で言えば、空間指向と言える。土地の概念があるのだ。座標 0,0 がホームであり、ここから同心円状に各自の土地が広がる。ホームに近いほど土地の価値も上がる。土地のメタファーは人間にとって馴染みやすく、覚えやすく、また退屈を凌げる効果もある。三次元ほどの豊かさはないが、三次元よりは軽いため、ボード（ワールド）を量産しやすい。たとえば毎週、新しいボードをつくって、オフィスがわりにしてもいい。週ごとにボードを見ながら振り返ることで、得られるものも多かろう。
