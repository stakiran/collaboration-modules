# PWEP
**PWEP** とは Powerpoint、Word、Excel、PDF の総称であり、コラボレーションの拙さを示す黄色信号とみなせる目安である。

コミュニケーションや情報共有においては、QWINCS やプレーンテキストなど、よりコラボレーティブに行える手段がすでに揃っている。にもかかわらず、古典的な PWEP に固執しているのは、自転車やバイクがあるのに車しか使わないようなものである。数百メートル以下、場合によっては数十メートルの近距離移動であっても毎回自動車を使うくらいに馬鹿げたことなのだ。

PWEP は正式な文書や高度な計算（Excelの場合）には重宝するが、その前段階の、組織内で完結するような大部分の仕事ではもはや不要である。PWEP を使っている、特に重用している組織があるとするなら、その時点で黄色信号なのである。
