name: save-to-memo
description: 直近の内容を markdown 形式で保存
metadata:
  author: stakiran
---

直近の出力を .memo/ 配下に markdown 形式で保存しなさい。

保存ファイル名が与えられた場合は、その名前を使いなさい。既存ファイルであっても上書きして良い。典型的には 1 や 2 など機械的にシンプルな名前が与えられる。

ファイル名が与えられなかった場合、既存のメモファイルを走査した上で、連番の続きとなる番号名を使いなさい。たとえば 1.md、2.md、4.md があったとしたら、3.md を使うはずだ。

メモファイルを書き込んだ後、そのメモファイルを開きなさい。ただし利用者は VSCode を使っているものとします。
